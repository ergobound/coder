Анализатор спектра для различных платформ SDR,
написанный на базе <a href="https://github.com/xmikos/qspectrumanalyzer" target="_blank">qspectrumanalyzer</a>

О преимуществах данной версии читайте далее.

**Общие улучшения:**  
 - Обновленая идеально выверенная формула для Average, которая не пропускает короткие всплески,
 а так же убирает лишний шум с графика. Совмещает два этапа усреднения:
 EMA по предыдущему во времени + EMA по частоте 
 - Профили скорости
 - Глобально изменены стили программы.

**Маркеры:**  
    Маркеры, обозначающие n-е количество пиков. Нестабильность графика может делать передвижение
маркеров очень хаотичными и их невозможно будет наблюдать человеческому глазу.
По этому важно сохранять точность, быстроту обновления маркеров и одновременно их плавность перехода.
Для достижения этого баланса используется:
 - EMA сглаживание по истории
 - Порог по уровню
 - Локальные максимумы
 - Минимальное расстояние между пиками
 - Квадратичная интерполяция вершины пика

**Мониторинг:**  
    Поиск частот превышающих абсолютное или среднее пороговое значение.
Имеет умную сортировку по любому из заголовков таблицы. 
 - Локальный максимум
 - Округление частот (Удобно, если очень много детализированных данных, объем которых необходимо уменьшить)
 - Экспорт в таблицу xlsx

**Цифровые сигналы:**  
    Адаптивные бары, отображающие самую высокую частоту из фиксированных диапазонов (Стандартов связи),
под каждым из которых есть водопад, в котором хранится история всех предыдущих пиков.
 - Переключатель Downlink/Uplink
 - Каждый бар можно отдельно включить/выключить


*Для запуска и использования программы необходимо приобрести лицензионный USB-накопитель.*
